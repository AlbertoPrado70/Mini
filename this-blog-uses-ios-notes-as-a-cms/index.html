This blog uses iOS Notes as a CMS

I love simplicity.

Simplicity allows me to focus on the important things.

Like writing a new blog.

What are you seeing is a blog made with eleventy that uses iOS Notes App as a CMSÂ 

How does it works?

I dont think that the setup is that complicated:

  • A Shortcut sends all modified notes from today to a special email address
  • A Github action triggers an Eleventy Build
  • Eleventy uses imap-flow to create data from received emails

This simple setup allows me to create posts from my notes.

The neat part is that i can update posts by modifing published posts, as long i dont change my permalink.

Markdown

This setups allows me to use markdown in my notes.

Another cool thing is that i can include other data to my posts, like the battery level at the time of posting.

Yeah, i know that is not usefull, but it is kinda cool.

Translating to english

English is not my first languaje so this setup allows me to check errors before sending the email.
