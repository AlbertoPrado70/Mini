This blog uses iOS Notes as a CMS

I like the iOS Notes app because it’s simple and available on all my devices.

Even tough i use WordPress for allmost all my sites, i cant resist the idea of using Notes as a CMS.

So, I made it happen!

What you’re seeing is a simple blog built with Eleventy, and all the post data comes directly from my phone.

How does it works?

I saw on hacker news an App here that does something similar to my setup, but maybe mine is even simpler.

A Shortcut sends all notes modified today to a special email address

  • A Github action triggers an Eleventy Build
  • Eleventy uses imap-flow to generate data from the received emails

This simple setup lets me create posts directly from my Notes.

The neat part is that i can update posts by modifing published posts, as long i dont change the permalink.

Markdown

Another cool trick of the Shortcuts apps is that i can convert Rich text to markdown.

Device data

Not as usefull as the markdown part, but i can include other data to my posts, like the battery level at the time of posting.

Yeah, i know that is not usefull, but it is kinda cool.
